# Guys, BFB-Minning-pool! Coin is BFBCoin.

I merged few pull requests and it supporting new nodejs versions.

Thank you.





-------
### Install
```
git clone https://github.com/BFBCOIN/BFB-Minning-pool/ pool
cd pool
npm install
node init.js
```
-------
### Requirements
* Node 8.x.x or higher
* Coin daemon
* Redis Server
